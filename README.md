# **Unos Hermanos**

## Это многопользовательская uno игра, созданная в рамках семестровой работы для предмета ОРИС(ОсновыРазработыИнформационныхСистем)

---

## Что она может?

* Возможность создавать лобби, которые независят друг от друга
* Создавать визуал для игрока, предоставляется 4 аватара не выбор, а также ввод никнейма
* Настройки игры, выбор игровых правил
* Игру от 2 до 4 человек

---

## Что по игре

### Хост создаёт лобби, где генерится случайный ключ, по-которому игроки уже могут присоединяться к лобби

### Хост может начать игру, даже, если не набралось максимальное кол-во игроков указанное в лобби

### Правила игры - стандартные для уно, с поправкой на расширение игровых правил
